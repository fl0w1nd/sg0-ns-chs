# NS平台命运石之门0汉化

## 汉化说明
* 移植官方中文文本
* 菜单设置同步官方 PC 汉化
* 部分游戏内 CG 同步官方 PC 汉化
* 过场动画暂未处理

## 使用说明
首先在右侧发行版下载汉化文本压缩包并解压
将压缩包内的`010037700E9B8000`文件夹放置于`atmosphere/contents`目录下

## 其他说明 
由于是针对官方英文的汉化，所以 NS 语言需要设置为日语以外的其他语言，因为 NS 的 sg0 是日英双语，而日语文本我未作任何替换处理
